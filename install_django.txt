1 # Запустите следующую команду в командной строке
 для установки virtualenv:

# pip3 install virtualenv

2 # Если в системе была предустановлена Python 2.X,
 а вы установили Python 3.X, то нужно указать virtualenv,
 чтобы он работал с последней версией.
просто запустите:

# virtualenv -p python3 my_env

Обновление после редактирования OP-х годов:

В версии OP virtualenv была ошибка, как описано здесь .
 Проблема была устранена путем запуска:

# pip install --upgrade virtualenv  # это не обязательно

3 # Используйте следующую команду для активации виртуальной среды:

# source my_env/bin/activate
# python -V # проверка версии 

4 # Обновление:

# pip install --upgrade virtualenv  # это не обязательно

5 # Используйте следующую команду в оболочке, чтобы установить Django
 с помощью pip:

# pip install Django==2.0.5

 
Отключить виртуальную среду можно с помощью команды 

# deactivate
