Запустите следующую команду в командной строке для установки virtualenv:

1 # pip3 install virtualenv

После установки virtualenv, создайте виртуальную среду
 с помощью следующей команды:

# virtualenv my_env # это не делай

Это создаст папку my_env вместе со средой Python.
 Любые библиотеки Python, установленные с активированной 
 виртуальной средой Python, будут установлены
 в папку my_env/lib/python3.7/site-packages.
 
Если в системе была предустановлена Python 2.X,
 а вы установили Python 3.X, то нужно указать virtualenv,
 чтобы он работал с последней версией.

просто запустите

2 # virtualenv -p python3 my_env

Обновление после редактирования OP-х годов:

В версии OP virtualenv была ошибка, как описано здесь .
 Проблема была устранена путем запуска:

5 # pip install --upgrade virtualenv  # это не обязательно

Используйте следующую команду для активации виртуальной среды:

3 # source my_env/bin/activate
4 # python -V  # проверка версии

Отключить виртуальную среду можно с помощью команды 

# deactivate
